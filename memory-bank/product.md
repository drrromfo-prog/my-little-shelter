# Product

My Little Shelter is a personal media logging site for keeping films, TV series, anime, documentaries, and books in one quiet place.

Core product intent:

- feel like a private cultural shelter, not a dashboard
- make adding a record easy
- keep browsing calm, readable, and emotionally expressive
- preserve personal context through notes, quotes, mood, and progress

Current supported categories:

- `movie`
- `tv`
- `anime`
- `documentary`
- `book`

Current supported user-facing statuses:

- 待看 / 待读
- 正在进行
- 已看 / 已读
- 搁置

Current core user actions:

- add a record manually
- autofill from a Douban link
- edit or delete a record
- search, sort, and filter
- switch between grid and list views
- browse quotes, mood calendar, and stats

The homepage should remain the clearest entry point and keep a visible “新增记录” action.
