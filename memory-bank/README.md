# My Little Shelter Memory Bank

This folder keeps a lightweight project memory for the current state of My Little Shelter.

Current live setup:

- GitHub repo: `https://github.com/drrromfo-prog/my-little-shelter`
- Railway production: `https://my-little-shelter-production.up.railway.app/`
- Healthcheck: `https://my-little-shelter-production.up.railway.app/healthz`

Contents:

- `product.md`: product scope and user goals
- `tech-stack.md`: runtime, frontend, tooling, deployment stack
- `architecture.md`: how the app is structured
- `routes-and-pages.md`: routes, views, and page responsibilities
- `data-model.md`: persisted fields and frontend-only fields
- `ui-style.md`: visual system and interaction tone
- `implementation-plan.md`: short next-step plan
- `progress.md`: completed work summary
- `current-focus.md`: current product and engineering focus
- `deployment.md`: GitHub + Railway deployment notes
- `known-issues.md`: factual known issues and constraints
- `changelog-ai.md`: AI-assisted change log

Use this folder as a fast orientation layer before making changes.
