# UI Style

Visual direction:

- warm cream / stone background
- soft editorial feeling
- rounded cards and panels
- subtle borders and shadows
- calm hierarchy instead of dashboard clutter

Core layout:

- fixed left sidebar
- right main content area
- homepage header with search, filters, and primary action
- card-first browsing

Interaction tone:

- restrained hover states
- no loud gradients
- no dark tech aesthetic
- no heavy motion

Current notable UI behaviors:

- grid / list view switching
- subtle ambient background color layer
- Color Thief based card accents with graceful fallback
- compact non-home subpages
- import/export visually downgraded to secondary actions
- empty states now include a direct primary add button

Recent simplification direction:

- remove low-priority views
- reduce explanatory text on secondary pages
- improve mobile first-screen clarity
- keep homepage as the strongest interaction surface
