Priority right now:

- keep the homepage strong and uncluttered

- improve mobile usability with minimal layout risk

- 信息精简

- 空状态直达新增

When making the next change, prefer:

- small reversible edits
- frontend-only polishing first
- no unnecessary API or schema churn
